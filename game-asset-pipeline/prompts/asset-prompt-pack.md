# Asset Prompt Pack

## Global Style Prefix

Clean low-poly browser game asset, orthographic isometric camera, strong readable silhouette, saturated fantasy accents, rough non-photoreal materials, simple geometry that can become a GLB, no background, centered object, marketplace-ready.

## Negative Prompt

Photorealistic, text, watermark, logo, cluttered background, extra limbs, melted shape, noisy texture, extreme thin wires, transparent glass overload, tiny unreadable details, gore, horror.


## Character Rigging Rule

For any humanoid, creature, wearable avatar, mount rider, NPC, enemy, boss, or animated pet that needs rigging, generate the asset in a clean T-pose by default:

`full body T-pose, arms extended horizontally, straight neutral legs, feet visible, symmetrical front view, simple open hands, joints unobstructed, rigging-friendly character sheet pose`

Avoid:

`action pose, bent elbows, crossed arms, cropped feet, cropped hands, weapon covering body joints, seated pose, crouch, extreme perspective, dramatic foreshortening`
